# Environment Variables Setup Guide for PawMatch

## Required Environment Files:

### 1. `.env` (Main development environment)
```bash
# Supabase Configuration
EXPO_PUBLIC_SUPABASE_URL=your_supabase_project_url_here
EXPO_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key_here

# OAuth Configuration
EXPO_PUBLIC_OAUTH_REDIRECT_URL=pawmatch://oauth/callback
EXPO_PUBLIC_AUTH_REDIRECT_URL=pawmatch://oauth/callback  
EXPO_PUBLIC_AUTH_MOBILE_REDIRECT_URL=pawmatch://oauth/callback

# Google OAuth Configuration (Optional)
EXPO_PUBLIC_GOOGLE_CLIENT_ID=your_google_client_id_here

# Mobile Configuration
EXPO_PUBLIC_MOBILE_SCHEME=pawmatch
EXPO_PUBLIC_USE_WEB_FLOW=false

# Admin Configuration
EXPO_PUBLIC_ADMIN_EMAIL=your_admin_email_here

# App Configuration
APP_ENV=development
APP_VERSION=1.0.0
DEBUG_MODE=true
```

### 2. `.env.local` (For admin panel - create in pawmatch-admin/ folder)
```bash
# Supabase Configuration (same as mobile app)
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url_here
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key_here

# Admin Configuration
ADMIN_EMAIL=your_admin_email_here
```

## Setup Instructions:

1. **Get Supabase Credentials:**
   - Go to https://supabase.com
   - Create a new project
   - Go to Settings > API
   - Copy the URL and anon key

2. **Configure Google OAuth (Optional):**
   - Go to Google Cloud Console
   - Create OAuth 2.0 credentials
   - Add redirect URI: your_supabase_url/auth/v1/callback

3. **Replace the placeholders:**
   - Replace `your_supabase_project_url_here` with your actual Supabase URL
   - Replace `your_supabase_anon_key_here` with your actual Supabase anon key
   - Replace `your_admin_email_here` with your admin email
   - Replace `your_google_client_id_here` with your Google Client ID (if using Google auth)

4. **Files to create:**
   - `/Users/fahim/PawMatch/.env` (main app)
   - `/Users/fahim/PawMatch/pawmatch-admin/.env.local` (admin panel)

## Important Notes:
- Never commit .env files to git
- Keep .env files in .gitignore
- Production builds will use the same .env file
- Admin panel needs its own .env.local file in the pawmatch-admin folder
